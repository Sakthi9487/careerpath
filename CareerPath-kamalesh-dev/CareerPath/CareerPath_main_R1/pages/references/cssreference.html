    Introduction

    CSS (Cascading Style Sheets) is the language used to style and layout web pages. This comprehensive reference guide includes all CSS properties, selectors, at-rules, and functions as documented by W3Schools.

    CSS Syntax

    css
    selector {
    property: value;
}

    Example:

    css
    p {
    color: red;
    text-align: center;
}

    CSS At-Rules

    At-rules are CSS statements that instruct CSS how to behave.

    Example Usage

    css
    /* Media - Responsive design */
@media screen and (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

/* Keyframes - Animation */
@keyframes slide {
    from {
        left: 0px;
    }
    to {
        left: 200px;
    }
}

/* Font Face - Custom font */
@font-face {
    font-family: myFont;
    src: url("myfont.woff");
}

/* Supports - Feature detection */
@supports (display: grid) {
    div {
        display: grid;
    }
}

    Color Names

    CSS supports 140 standard color names. Examples:

    • red, blue, green, yellow
    • black, white, gray
    • orange, purple, pink
    • brown, cyan, magenta

    RGB / RGBA

    css
    /* RGB */
color: rgb(255, 0, 0);     /* Red */
color: rgb(0, 255, 0);     /* Green */
color: rgb(0, 0, 255);     /* Blue */

/* RGBA (with alpha/opacity) */
color: rgba(255, 0, 0, 0.5); /* Red with 50% opacity */

    HEX

    css
    color: #ff0000;  /* Red */
color: #00ff00;  /* Green */
color: #0000ff;  /* Blue */
color: #000;     /* Black (shorthand) */
color: #fff;     /* White (shorthand) */

    HSL / HSLA

    css
    /* HSL */
color: hsl(0, 100%, 50%);    /* Red */
color: hsl(120, 100%, 50%);  /* Green */
color: hsl(240, 100%, 50%);  /* Blue */

/* HSLA */
color: hsla(0, 100%, 50%, 0.5); /* Red with 50% opacity */

    Special Values

    css
    color: currentColor; /* Current color value */
color: transparent;  /* Fully transparent */
color: inherit;      /* Inherits parent value */
color: initial;      /* Default value */
